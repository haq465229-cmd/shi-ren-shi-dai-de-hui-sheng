# TIERIA Homepage Project

这个文件夹已经按用途分成三部分：

- `APP`: 可运行的 Vite + React 项目。进入这个文件夹后可以运行 `npm.cmd run dev` 本地预览，或运行 `npm.cmd run build` 生成发布文件。
- `Design`: 设计素材与参考图。`Screenshots` 是原项目根目录下的页面效果图，`Assets` 是页面使用的视觉素材副本。
- `Docs`: 文档、日志和独立预览页。`打开首页.html` 已调整为从 `../APP/src/assets/` 读取图片资源。

原始项目目录没有被删除，当前目录是整理后的迁移版本。
